OB降臨システム webサービス
========================

Dependencies
------------

* sinatra
* [saykana](http://www.a-quest.com/quickware/saykana/)


Run Server
----------

config

    % cp sample.config.yaml config.yaml

then, edit it.


Instlal Dependencies

    % sudo gem install bundler
    % bundle install


run

    % ruby development.rb
